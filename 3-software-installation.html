How to use Adafruit TSL2591 Light Sensor with a Raspberry Pi

3 Software installation

Published: Fr 16 Dezember 2022
By BrandnerN

In Software installation.

This is tested on Ubuntu 20.04 LTS

Information of the manufacturer:

repo
docs

Install Adafruit library and driver bundle via

pip3 install adafruit-circuitpython-lis3dh
(see https://github.com/adafruit/Adafruit_CircuitPython_Bundle for more information).

Istall system-wide driver for TSL

sudo pip3 install adafruit-circuitpython-tsl2591
(see https://docs.circuitpython.org/projects/tsl2591/en/latest/index.html for more information).

Activate I2C-GPIO

sudo raspi-config in terminal of the pi
Go to "3 Interface Options"
Select "P5 I2C"

Clone this repository on local machine (for Linux-users preferably in home directory)

git clone https://github.com/kafu16/rpi_lightsensor.git
If not cloned in home directory: Adapt the path in rpi_lightsensor/measure_plot.sh line 5 that points to the directory where you cloned the repository (adapt the variable local_repo).

Parts of the script are adapted from a usage example that can be downloaded here.

Give the script exection permission
cd rpi_lightsensor
chmod +x measure_plot.sh

On the RPi clone the same repository in the home directory. You should get a directory /home/pi/rpi_lightsensor.

Activate ssh on the rpi

On rpi go to applications menu (top left)/Raspberry Pi Configuration/Interfaces
Select "Enable"
